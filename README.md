# ABSOLUTE CINEMA

An AI-powered narrative analysis platform that analyzes stories, identifies cinematic influences, generates narrative DNA, evaluates originality, and produces professional consultation reports.

---

## Features

### Story Analysis
- Movie influence detection
- Genre prediction
- Similarity scoring
- Top match ranking

### Narrative DNA
- Structure analysis
- Tone analysis
- Pace analysis
- Conflict analysis
- Ending analysis
- Protagonist arc analysis

### Story Intelligence
- Story Scorecard
- Character Profile
- Theme Analysis
- Originality Report
- Consultant Feedback
- Story Blueprint

### Visualization
- Influence breakdown
- PulseGraph
- Scorecard metrics
- Blueprint roadmap

### Reporting
- Professional PDF export
- Multi-page consultation reports

## Installation

### Backend

```bash
cd backend
.\venv\Scripts\activate
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
